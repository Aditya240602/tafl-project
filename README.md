## 📌 Project Description

This project involves building a web application that generates derivation sequences and parse trees for strings derived from a given context-free grammar (CFG).

The user inputs grammar rules along with a string, and the system generates step-by-step derivations such as **leftmost** and **rightmost derivations**.

Additionally, the application produces a **graphical parse tree** that visually represents how the string is derived from the grammar.

This tool helps students understand:
- Grammar derivations  
- Structure of context-free languages  
- Parse tree construction

## 🚀 How to Use

1. Enter grammar rules (one per line)  
2. Enter the input string  
3. Choose:
   - Leftmost Derivation  
   - Rightmost Derivation  
   - Generate Parse Tree  
4. View results instantly
