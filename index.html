Quick Intro

Context-Free Grammar, Derivation, and Parse Tree

A context-free grammar (CFG) defines how strings are formed using production rules. A derivation shows how the start symbol is rewritten step by step into the target string. A parse tree is the structural view of the same process, where each expansion becomes child nodes in a tree.

Leftmost derivation: always expand the leftmost nonterminal first.

Rightmost derivation: always expand the rightmost nonterminal first.

Parse tree: records the hierarchy of expansions from root (start symbol) to terminal leaves.
